# 学术推广方法论

## 核心原则
1. **循证为基础**：所有推广主张必须有证据支持
2. **合规为底线**：不超适应症、不夸大、不误导
3. **差异化为抓手**：找到产品的临床差异化定位
4. **数据说话**：用数字表达，避免模糊表述

## 学术推广金字塔

```
         ⬆️ 价值主张
        /   \
    🎯 定位  🎯 区别
   /   \      /   \
📊 疗效 📊 安全 📊 便捷
/ \    / \    / \
文献 · 指南 · 共识 · RWE
```

## 不同场景的推广策略

### 科室会
- **目标**：教育+影响处方决策
- **内容**：疾病负担 → 治疗痛点 → 产品证据 → 临床价值
- **时长**：20-30分钟 + 10-15分钟Q&A
- **频率**：每季度至少1次重点科室

### 一对一KOL沟通
- **目标**：建立学术共识+KOL背书
- **内容**：最新数据分享 + 研究合作探讨
- **关键**：倾听KOL观点，寻找学术共鸣点

### 学术会议
- **展台**：简洁有力，突出关键数据
- **卫星会**：权威讲者+高质量内容
- **壁报**：清晰的研究设计和结果

### 代表推广话术
- **开场**：了解医生治疗痛点
- **数据锚点**：1-2个最有说服力的数据
- **差异化**：与现有方案的区别
- **行动号召**：建议尝试的场景

## 合规检查清单
- [ ] 所有数据标注来源（文献PMID/会议年份）
- [ ] 不含超适应症内容
- [ ] 不使用"最好""唯一""首创"等绝对化用语
- [ ] 竞品对比使用客观数据而非主观评价
- [ ] 不良反应信息完整披露
- [ ] 材料经过医学合规审核
